Задание 8. Контрольное задание по теме "Применение XML, особенности различных технологий".

В исходном файле находятся данные об играх компании "Alawar". Все игры разбиты по категориям
("стрелялки", "бегалки" итп). Требуется написать XSL-преобразование для вывода этого каталога 
игр в красивом иs удобном для конечного пользователя виде:

	на основании содержащихся в файле данных на экране должно автоматически формироваться 
	навигационное меню из категорий игр;
	
	при наведении мышкой или клике на конкретную категорию должны выводиться игры этой категории
	в виде списка;
	
	для каждой игры необходимо выводить название, иллюстрирующую картинку и краткое описание; 
	при клике на любой из этих элементов должен осуществляться переход на страницу игры.

Допускается использование JavaScript для выборки и манипулирования данными. Для оформления использовать CSS.

Исходные данные: games.xml
