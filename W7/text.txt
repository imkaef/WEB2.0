Задание 7. Создание интерактивных страниц с помощью XML и XSL(T).

В исходном файле находятся данные о телепередачах телеканала "Наука" на определенные даты. 
Требуется написать XSL-преобразование для вывода телепрограммы в красивом и удобном для конечного пользователя виде:

	на основании содержащихся в файле данных на экране должно автоматически формироваться навигационное меню из дат, 
	на которые есть данные о телепередачах;

	при наведении мышкой или клике на конкретный день должны выводиться передачи этого дня в виде списка; элемент 
	списка состоит из времени и названия передачи.

Допускается использование JavaScript для выборки и манипулирования данными. Для оформления использовать CSS.
